# Maragi

## About

Maragi is a Direct2D UI based twitter client which supports multi-column, multi-user, streaming.

## Compilation

1.  Prepare Perl (ActivePerl, Strawberry Perl, â€¦. anything is ok), Visual Studio 2012, [Visual C++ Compiler November 2012 CTP](http://www.microsoft.com/en-us/download/details.aspx?id=35515) and Windows SDK.

2.  Fill Maragi/Tokens.txt as

        $1=<consumer key>
        $2=<consumer secret>

3.  Open Maragi.sln and Build.

4.  Copy bin&lt;platform bits&gt;/*.dll into a directory contains an executable file.

5.  Enjoy!
